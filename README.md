# MyFileManager
Основные команды

"cd path"      // path - путь к каталогу, показ всех каталогов и файлов, находящихся по указанному пути
"select path"  // path - путь к файлу или каталогу, выбрать файл или каталог для работы (копирование/удаление)


Команды для работы с файлами/каталогами
Для работы с файлами и каталогами их сначала нужно выбрать (команда select);

"copy path"   //path - путь с конечным названием куда будет скопирован файл/каталог
"delete"      //удаление выбранного файла или каталога
